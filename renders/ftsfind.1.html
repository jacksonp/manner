FTSFIND

ftsfind - search for files in a directory hierarchy

SYNOPSIS

ftsfind [-H] [-L] [-P] [-D debugopts] [-Olevel] [path...] [expression]

DESCRIPTION

The ftsfind and find programs are functionally identical. The only difference between them is the implementation internals of how the filesystem is searched. The find binary has a recursive implementation while ftsfind uses fts(3) The fts-based implementation will replace the recursive implementation but for the moment both implementations are installed.

The implementation which is installed as find on this system is the recursive implementation, since the --without-fts option was passed to the configure script when findutils was built.

SEE ALSO
