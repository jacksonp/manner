dpkg-vendor

dpkg-vendor - queries information about distribution vendors

SYNOPSIS

dpkg-vendor [option...] command

DESCRIPTION

dpkg-vendor\fP is a tool to query information about vendors listed in /etc/dpkg/origins\fP. /etc/dpkg/origins/default\fP contains information about the current vendor.

COMMANDS

--is vendor
Exits with 0\fP if the current vendor is vendor\fP. Otherwise exits with 1\fP.
--derives-from vendor
Exits with 0\fP if the current vendor distribution is a derivative of vendor\fP, otherwise exits with 1\fP. It uses the Parent\fP field to browse all ancestors of the current vendor.
--query field
Print on standard output the value of the vendor-specific field\fP for the current vendor.
--help
Show the usage message and exit.
--version
Show the version and exit.

OPTIONS

--vendor vendor
Assumes the current vendor is vendor\fP instead of discovering it with the DEB_VENDOR\fP environment variable or /etc/dpkg/origins/default\fP.

ENVIRONMENT

DEB_VENDOR\fP
This setting defines the current vendor. If not set, it will discover the current vendor by reading /etc/dpkg/origins/default\fP.
